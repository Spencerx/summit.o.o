Open Build Service (OBS): Facts, Features and Future

Dr. SCHRöDER, Michael and SCHRöTER, Adrian - SUSE

PRESENTATION - Bonaire 5, Saturday, 4:15 p.m. to 5:00 p.m.

Since the start of the project in the end of 2005, the Open Build Service (OBS) has become a versatile solution for package building used in numerous instances.

This talk will give you a short overview over the main featues of the build service like support of multiple package formats and distributions, automatic package rebuilding, a versatile project setup system, collaboration workflow support, the OBS interconnect, and many more.

It will then proceed to showing how openSUSE is using OBS to create and maintainer the openSUSE distribution.

Finally it will give you an overview of the features we're currently working on, our roadmap for the next releases and some ideas for the far future.
