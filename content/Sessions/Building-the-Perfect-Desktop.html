--- title: Building the Perfect Desktop Solution - openSUSE ---

Best practices for Bugzilla and bug reporting

SICKENBERG, Holger - SUSE

PRESENTATION - Bonaire 6, Sunday, 9:15 a.m. to 10:00 a.m.

If you are not a software developer, reporting a bug is the easiest way to improve openSUSE. But how to report a bug and how to deal with openSUSE's bug tracking tool "Bugzilla"? This talk should give users and developers an idea about how to use Bugzilla to report bugs and how to use Bugzilla to work on bugs.
